Instant

About

Instant is a fully functioning instant messaging mobile application. Similar like WhatsApp or Telegram, Instant uses phone number authentication to login. This is done using Google’s Firebase Authentication package, which handles sending verification codes to valid phone numbers. Instant was created using Flutter, an open-source UI software development kit, in order to develop for both IOS and Android in one code base. To model data Instant uses Google’s Firebase Firestore, a NoSQL real-time database. This database provides many conveniences such as automatic scalability, file storage, and various kinds of authentications. Instant pushed my limits in system design, and database modeling.

Technologies
  • Flutter
  • Dart
  • Firebase
